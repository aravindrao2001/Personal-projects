# IT Setup Guide

## Laptop Request

New employees receive a company laptop during onboarding.

Laptop requests must be submitted through the IT portal.

## VPN Access

Employees who work remotely must connect to the company network through
the VPN.

VPN access requires approval from the employee's manager and IT
administrator.

## Software Installation

Employees may install approved productivity tools such as Slack, Zoom,
and Microsoft Office.

Administrative permissions are required to install development tools.
